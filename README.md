Выполнено в Unity 2020.3.47f1. Содержит 2 сцены:

- Простой экран торговли

- Демонстрацию реализации сервиса поиска пути. Для визуализации используется Gizmos, убедитесь что в редакторе включено их отображение (и там же отключите отображение canvas). <br>
Ввиду специфики данного инструмента рекомендуется запускать сцену в низком разрешении.
